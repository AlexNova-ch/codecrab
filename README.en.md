## Core Concept

Mobile Claude Code GUI client that eliminates the need for terminal:

- Connect to a remote Claude Code instance running on a server
- View task execution progress (streaming output)
- Approve/deny Claude Code's operation requests (permission confirmation)
- Browse file tree and view file contents
- Start new tasks and switch projects
