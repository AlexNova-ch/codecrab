## 核心概念

手机端 Claude Code GUI 客户端，让你无需终端即可：

- 连接远程服务器上运行的 Claude Code 实例
- 查看任务执行过程（流式输出）
- 审批/拒绝 Claude Code 的操作请求（permission confirmation）
- 浏览文件树、查看文件内容
- 发起新任务、切换项目
